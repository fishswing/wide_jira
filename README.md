## Wide JIRA ##

A simple user script to widen the "description" box of JIRA

URLs matching `http(s)://jira.*` or `http(s)://*/jira/` will auto apply the script;
For other URLs, add it to "User Includes" in your UserScript manager.

&nbsp;
